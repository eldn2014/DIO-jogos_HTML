*Formatadores de textos

usando a tag strong deixa o texto em negrito
usando a tag b deixa o texto em negrito

quando falamos de programação simples as tags b e strong são identicas, porem semanticamente
falando e pensando na acessibilidade, quando queremos deixar um texto em negrito usamos o strong.


usando a tag i deixa o texto em italico
usando a tag u deixa o texto sublinhado
usando a tag sub deixa o texto menor e alocado na parte inferior
usando a tag mark deixa o texto grifado
usando a tag sup deixa o texto maior e alocado na parte superior

as tags sup e sub são muito usandas em representação de numeros cientificos
exemplo: 10 2, 103

a tag blockquote é usada para referenciar citações de texto,
ela deixa um espaçamento do lado esquerdo.

usando a tag font por si so ela não faz nada, pois ela precisa de atributos
esses atributos são:

color: define a cor do texto
face: fonte do texto(formatação trebuchet se não achart formata arial)
face: fonte do texto(formatação arial)

*Div e Span

a div é muito usanda em estruturação de sites, onde servem de esqueleto, para manipular com acestividade no CSS é aconselhavel que cada div possua um id ou class.
a span diferente da div não é display block, ela não quebra linha, muito utiel para manipular trecho de textos

*Fieldset

a tag fieldset trabalha em conjunto com a tag formulario, muito usada de forma estutural,
ela é uma caixinha que mostra no display do usuario, ela envolve o formulario que esta dentro dela

*Embeds

o embed é uma tag que esta entrando em desuso nos sites criados recentemente, ela serve para por links de videos imagens e etc.
a mesma caiu em desuso devido a evolução da linguagem, que surgiu novas tags.

*Iframes

o iframe temos que ter cuidado com a questão seguraça, o iframe permite criar uma janela dentro do seu site,
onde o usuario pode navegar em outro site dentro dessa janela.
